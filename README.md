# Retail Demand Forecasting & AI Analytics Platform

## Project Overview

This project aims to build an end-to-end retail analytics platform using historical Walmart sales data. The system will analyze sales trends, perform demand forecasting using machine learning, and present business insights through an interactive Power BI dashboard.

---

## Objectives

- Analyze historical retail sales
- Perform exploratory data analysis (EDA)
- Build demand forecasting models
- Compare ML model performance
- Generate business insights
- Develop an interactive Power BI dashboard
